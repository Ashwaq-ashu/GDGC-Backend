# GDGC-Backend

this is the official Backend repository for the GDGC Backend 

- This contains the core functionalities for events and form creating for the GDGC website as well as the auth for the admin panel
- this is an open source repo so you are open to contibute , make sure to follow the MVC pattern
- please follow this video to understand MVC pattern in express JS : https://youtu.be/JLtXoru-ipo

Technologies : 
- Javascript
- ExpressJS
- MongoDB
